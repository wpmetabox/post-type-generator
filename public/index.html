WordPress Custom Post Type Generator

Generate custom post types for WordPress in a few click. Ready to copy and paste PHP code for custom post types.

Welcome to Meta Box Post Type Generator. This tool helps you to generate code for custom post types to use in your WordPress websites.

  • If this is the first time you use the tool, please read this guide.
  • If you want to use the generator inside WordPress admin area, please use Meta Box Lite, a single plugin that combines Meta Box, Meta Box Builder and all free extensions to manage post types, taxonomies, custom fields, and relationships with UI.
Intro

We created Meta Box in 2010 to help developers to create custom meta boxes faster and easier. Now, Meta Box is not only a library with powerful API for custom fields, but also a framework that helps you control your data the way you want.
